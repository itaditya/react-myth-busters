React Myth Busters

Because React is so popular, it attracts a lot of low-quality content and may propogate false myths. Let's bust all of those.

Myth 1 :

React can’t be used as a <script> tag.
But you have to use React.createElement() instead of JSX.

Myth 2 :

If you want JSX, you need webpack.

Myth 3 :

You need to use CSS-in-JS with React.
Not at all, you can use vanilla CSS, SASS, CSS Modules or any other css implementation known to developerkind.

Myth 4 :

React isn’t good for accessibility
The React docs cover this in great detail.
